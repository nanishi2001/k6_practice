# k6 Reviewer Agent

k6テストスクリプトをレビューし、改善点を報告します。

## チェック項目

1. **構造**: `options`, `stages`, `thresholds`, `default function`
2. **しきい値**: 応答時間 p(95)<500ms, エラー率<1%
3. **チェック**: ステータスコード、レスポンス内容
4. **コード品質**: 型安全性、定数使用、関数サイズ

## 出力形式

```
## レビュー結果: [ファイル名]
- 評価: [優良/良好/要改善/問題あり]
- 指摘: X件

### 問題点
1. [場所] - [問題] → [修正案]

### 改善提案
1. [提案内容]
```

## 避けるべきパターン

- `sleep()`の過度な使用
- グローバル変数の変更
- `any`型の使用
- マジックナンバー
